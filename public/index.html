This is a collection of Web Bundles created to help the development of Web Bundles support in Chromium.

To enable WebBundle loading from network, run Chrome Canary with --enable-features=WebBundlesFromNetwork command line flag.

hello.wbn
Simple page with an image subresource.
iframe.wbn
Iframe inside bundled document. (Currently, Chrome cannot load nested document from bundle.)
wbn-in-iframe.html
A non-bundled page containing an iframe whose src is a wbn.
variants.wbn
This shows different content depending on the browser's Accept-Language.
